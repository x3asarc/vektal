## Gate
Interpretation rules:
1) If default /health remote_ip is not expected server IP, DNS/routing is wrong.
2) If default fails but --resolve IPv4 passes, client resolver path is stale/cached.
3) If both default and --resolve fail, origin/nginx/frontend stack is unhealthy.
4) If IPv6 path shows old host while IPv4 is correct, stale AAAA is the cause.
