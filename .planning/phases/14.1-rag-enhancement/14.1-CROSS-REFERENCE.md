## What Phase 13.2 + 14 Built (Foundation)

The existing infrastructure provides a solid foundation:

1. **Neo4j runtime** with Docker, health checks, env wiring
2. **Entity contracts** (Pydantic v2) for all codebase entities
3. **Graph client singleton** with fail-open, timeout-bounded queries
4. **Episode ingestion pipeline** with Celery async emission
5. **Full codebase scanner** (Python AST, 1000+ files)
6. **Vector embeddings** (384-dim, cosine similarity, Neo4j index)
7. **Query templates** for common patterns (imports, callers, similar files)
8. **Incremental sync** via git hooks + consistency daemon
9. **Planning doc linking** (auto + natural reference detection)
10. **Intent capture** API for LLM-generated code
11. **Similarity detection** framework (5-tier classification)
12. **Oracle adapter** with fail-open evidence queries

## What the YAML Blueprint Adds (Missing Layer)

The missing 18 features cluster into 5 implementation themes:

1. **MCP Tool Server** (3 tools) - Expose graph to Claude Code
2. **Memory Engineering** (8 items) - Decision/Convention/BugRootCause nodes + relationships
3. **Search-then-Expand Bridge** (3 items) - Orchestrated vector→graph retrieval
4. **Semantic Caching** (2 items) - Cache layer with 0.92 threshold
5. **Operational Intelligence** (2 items) - Convention checking + discrepancy flagging
