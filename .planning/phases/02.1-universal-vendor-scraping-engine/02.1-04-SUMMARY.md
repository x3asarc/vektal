## Decisions Made

**DuckDuckGo HTML endpoint for web search:**
- Free service with no API key required
- Simple HTML parsing with regex (no heavy dependencies)
- Sufficient for vendor discovery fallback use case

**Six niche categories with incompatibility rules:**
- arts_and_crafts: incompatible with automotive, electronics
- automotive: incompatible with arts_and_crafts, fashion, food_beverage
- electronics: incompatible with arts_and_crafts, fashion, food_beverage
- fashion: incompatible with automotive, electronics
- home_garden: related to arts_and_crafts and fashion
- food_beverage: incompatible with automotive, electronics

**Strict vs flexible mode:**
- Strict mode (vendor_scope="focused"): Incompatible niches get confidence=0 (rejected)
- Flexible mode: Incompatible niches get confidence*0.2 (penalized but allowed)
- Unknown vendor niche: confidence*0.7 (30% penalty for caution)

**Store context in every search query:**
- CRITICAL: Never search SKU alone (prevents wrong vendor matches)
- Include top 3 keywords from store profile
- Include top 2 known vendors
- Example: "R0530 decoupage rice paper craft ITD Collection" instead of "R0530"
