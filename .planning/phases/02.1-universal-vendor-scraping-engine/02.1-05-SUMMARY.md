## Decisions Made

**Local-first AI strategy:**
- sentence-transformers runs locally (no API calls) for 80%+ of queries
- 120MB model (all-MiniLM-L6-v2) loads lazily on first use
- LRU cache (1000 entries) makes repeated queries instant

**API fallback design:**
- OpenRouter API only called when local confidence <0.85
- Gemini Flash chosen for cost efficiency ($0.10/1M tokens)
- API results cached (500 entries) to prevent repeated calls
- Niche mismatch detected and penalized (50% confidence reduction)

**Multi-stage pipeline:**
- Stage 1: Local pattern matching (instant, >=0.90 exits)
- Stage 2: Web search with context (free, >=0.80 exits)
- Stage 3: Local LLM classification (free, >=0.85 exits)
- Stage 4: API LLM inference (paid, final fallback)

**Confidence thresholds:**
- Pattern matching: 0.90+ for auto-proceed
- Web search: 0.80+ for auto-proceed
- Local LLM: 0.85+ for auto-proceed
- Confirmation required: <0.70
