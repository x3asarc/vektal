## Goal

Optimize episode ingestion by **batching episodes** instead of emitting one-by-one:
1. Replace individual `.delay()` calls with Celery `group()` pattern
2. Chunk large episode batches (>50) to prevent worker overload
3. Reduce Celery overhead by 40-50%

**Performance target**: 100 episodes ingested in ~5s (vs ~15s currently)

---

## Background

**Tool Calling 2.0 Pattern**: "Programmatic Execution" applied to background jobs - batch operations at infrastructure layer, not just API layer.

**Current bottleneck**:
- `emit_episode.delay()` called individually for each episode
- 100 episodes = 100 separate Celery tasks
- Each task: serialization + Redis write + worker pickup overhead (~150ms)
- Total: 100 × 150ms = 15 seconds Celery overhead

**After batching**:
- `group([emit_episode.s(ep) for ep in episodes]).delay()`
- 1 Celery group task containing 100 subtasks
- Group overhead: ~500ms (vs 15s sequential)
- **Speedup**: 30× faster Celery dispatch

**Source**: [Celery Canvas Primitives - Groups](https://docs.celeryq.dev/en/stable/userguide/canvas.html#groups)

### Step 2: Add Batch Emission Task

```python
# src/tasks/graphiti_sync.py

from celery import group, chord
from src.tasks.celery_app import celery_app

@celery_app.task(
    bind=True,
    max_retries=2,
    default_retry_delay=5
)
def emit_episode(self, episode: dict):
    """Emit single episode (existing task - unchanged)."""
    from src.jobs.graphiti_ingestor import GraphitiIngestor

    ingestor = GraphitiIngestor()
    try:
        ingestor.ingest_episode(episode)
    except Exception as e:
        logger.error(f"Episode ingestion failed: {e}")
        raise self.retry(exc=e)


@celery_app.task
def emit_episodes_batch(episodes: list[dict]):
    """Emit multiple episodes using Celery group pattern."""
    if not episodes:
        return {"status": "no_episodes"}

    # Chunk episodes (max 50 per chunk to prevent worker overload)
    CHUNK_SIZE = 50
    chunks = [episodes[i:i+CHUNK_SIZE] for i in range(0, len(episodes), CHUNK_SIZE)]

    results = []
    for chunk in chunks:
        # Create group of emit_episode tasks
        job = group([emit_episode.s(ep) for ep in chunk])

        # Execute group
        result = job.apply_async()

        # Wait for chunk completion (with timeout)
        try:
            chunk_results = result.get(timeout=30)  # 30s max per chunk
            results.extend(chunk_results)
        except Exception as e:
            logger.error(f"Chunk failed: {e}")
            # Continue with next chunk (don't fail entire batch)

    return {
        "status": "completed",
        "total_episodes": len(episodes),
        "chunks": len(chunks),
        "results": results
    }
```

### Step 4: Graphiti Ingestor Batch Support (If API Available)

If Graphiti has batch API, add wrapper:

```python
# src/jobs/graphiti_ingestor.py

class GraphitiIngestor:
    def ingest_episodes_batch(self, episodes: list[dict]) -> dict:
        """Ingest multiple episodes in one call (if Graphiti supports it)."""
        client = get_graphiti_client()

        if not client or not episodes:
            return {"status": "skipped", "reason": "no_client_or_episodes"}

        # Check for batch API
        if hasattr(client, 'add_episodes'):
            try:
                # Use native batch API
                result = client.add_episodes(episodes, timeout=30.0)
                return {"status": "success", "method": "native_batch", "count": len(episodes)}
            except Exception as e:
                logger.error(f"Batch ingestion failed: {e}")
                raise

        # Fallback: chunked loop
        results = []
        for episode in episodes:
            try:
                self.ingest_episode(episode)
                results.append({"status": "success"})
            except Exception as e:
                logger.error(f"Episode ingestion failed: {e}")
                results.append({"status": "error", "error": str(e)})

        return {
            "status": "completed",
            "method": "fallback_loop",
            "total": len(episodes),
            "successful": sum(1 for r in results if r["status"] == "success"),
            "failed": sum(1 for r in results if r["status"] == "error")
        }
```

## Edge Cases & Validation

1. **Empty episodes list**: Return `{"status": "no_episodes"}` (not error)
2. **Single episode**: Use individual `emit_episode.delay()` (no group overhead)
3. **Chunk failure**: Log error, continue with next chunk (don't fail entire batch)
4. **Graphiti client unavailable**: Gracefully skip (existing fail-open behavior)
5. **Timeout**: Individual episode timeout = 5s; chunk timeout = 30s; batch timeout = unlimited
