# Task 14.2-03: Deferred Loading + schema_json Column

**Status**: PLANNED
**Wave**: 2 (Infrastructure)
**Depends on**: 14.2-02 (Tool nodes must exist for discovery)
**Duration estimate**: 45-60 minutes
**Risk level**: LOW (Alembic migration + config flag)

---

## Goal

Enable **deferred tool loading** by:
1. Adding `schema_json` column to `AssistantToolRegistry` for persistent schema storage
2. Setting `deferred_loading: true` in `.claude/settings.local.json`
3. Initially exposing only `search_tools` to Claude; other tools discovered on-demand

**Why**: At 15+ tools, schema overhead becomes 1,200+ tokens/request. Deferring reduces this to ~80 tokens (only `search_tools` schema loaded initially).

---

## Background

**Tool Calling 2.0 Pattern**: "Deferred Loading" - Hide tool schemas until needed. Expose only a discovery mechanism (`search_tools`). Claude queries it when intent requires a tool not in the base set.

**Current bottleneck**:
- 9 tools total (3 MCP + 6 assistant)
- ~720 tokens/request schema overhead
- Phase 15 adds 5+ more tools (refactor, optimize, pattern-detect)
- At 15 tools: 1,200+ tokens/request

**After deferral**:
- Only `search_tools` schema loaded (~80 tokens)
- Other schemas fetched via `search_tools("update prices")` query
- Token overhead scales with *used* tools, not *available* tools

**Source**: [MCP Best Practices - Deferred Loading](https://modelcontextprotocol.info/docs/best-practices/)

---

## Current State

**Tool storage**:
- MCP tools: Hardcoded in `mcp_server.py`
- Assistant tools: `assistant_tool_registry` table with `metadata_json` (no `schema_json`)

**Schema loading**:
- All tools loaded upfront via `list_tool_contracts()` (MCP) and `project_tools_for_tier()` (assistant)
- No conditional loading mechanism

**Gap**: No persistent schema storage for assistant tools. No deferred loading config.

class AssistantToolRegistry(db.Model):

### Step 4: Conditional Loading in `mcp_server.py`

Modify `list_tool_contracts()` to respect deferral flag:

```python
# src/graph/mcp_server.py

def list_tool_contracts() -> list[dict]:
    """Return tool schemas. If deferred_loading=true, return only base tools."""
    import json
    from pathlib import Path

    # Load config
    config_path = Path(__file__).parent.parent.parent / ".claude" / "settings.local.json"
    config = json.loads(config_path.read_text()) if config_path.exists() else {}

    mcp_config = config.get("mcp_server", {})
    deferred = mcp_config.get("deferred_loading", False)
    base_tools = set(mcp_config.get("base_tools", ["search_tools"]))

    # All tools
    all_tools = [
        _query_graph_schema(),
        _get_dependencies_schema(),
        _retrieve_intent_schema(),
        _batch_query_schema(),      # from 14.2-04
        _search_tools_schema()       # from 14.2-02
    ]

    # Filter by deferral setting
    if deferred:
        # Only return tools in base_tools list
        return [t for t in all_tools if t["name"] in base_tools]

    # Threshold check: if few tools, load all anyway
    threshold = mcp_config.get("discovery_threshold", 10)
    if len(all_tools) < threshold:
        return all_tools

    # Default: return all
    return all_tools
```

### Step 5: Tool Schema Retrieval in `search_tools`

Enhance `search_tools` to return full schemas (not just names):

```python
# src/graph/mcp_server.py (in _search_tools_handler)

async def _search_tools_handler(query: str, tier: int | None, top_k: int) -> dict:
    """Search Neo4j for tools, return full schemas for deferred loading."""
    from src.graph.query_interface import execute_template_query

    results = await execute_template_query(
        template_name="tool_search",
        params={"query_embedding": get_embedding(query), "tier": tier, "top_k": top_k}
    )

    tools = []
    for record in results:
        # Return FULL schema for immediate use
        schema = json.loads(record["schema"])
        tools.append({
            "name": record["name"],
            "description": record["description"],
            "inputSchema": schema.get("inputSchema"),
            "input_examples": json.loads(record["examples"]) if record["examples"] else [],
            "relevance_score": record["score"],
            # Include full schema for Claude to "load" the tool
            "fullSchema": schema
        })

    return {
        "tools": tools,
        "query": query,
        "tier_filter": tier,
        "usage_hint": "Use fullSchema to call these tools in subsequent requests"
    }
```

**File modified**: `src/graph/mcp_server.py`
**LOC**: +10 lines (enhance existing handler)
