## Goal

Integrate **external research capabilities** for Phase 15 self-healing agents:
1. **Firecrawl MCP** - Web scraping/crawling for vendor documentation
2. **Perplexity API** - AI-powered web search as fallback
3. Fallback cascade: Firecrawl → Perplexity → Local search
4. Register as discoverable tools in Neo4j

**Use case**: Phase 15 agents discovering vendor API changes, researching error patterns, finding documentation.

---

## Background

**Why both tools**:
- **Firecrawl**: Structured web crawling, respects robots.txt, Markdown output
- **Perplexity**: AI search with citations, better for conceptual queries

**Fallback pattern**:
```
Query: "How to authenticate with Pentart API in 2026"
  ├─ Try Firecrawl (vendor site scrape) ──> Success → Return
  ├─ Fallback to Perplexity (AI search) ──> Success → Return
  └─ Fallback to Local (graph search)    ──> Return cached/empty
```

**Integration point**: Phase 14.2 tool discovery infrastructure (14.2-02) already supports external tools via `ToolNode` entities.

- [Firecrawl MCP Server](https://github.com/modelcontextprotocol/servers/tree/main/src/firecrawl)

### Step 2: Create Perplexity API Client

Perplexity doesn't have native MCP server - create wrapper:

```python
# src/integrations/perplexity_client.py

import httpx
import os
from functools import lru_cache

class PerplexityClient:
    """Wrapper for Perplexity AI API."""

    def __init__(self, api_key: str | None = None):
        self.api_key = api_key or os.getenv("PERPLEXITY_API_KEY")
        self.base_url = "https://api.perplexity.ai"
        self.rate_limit = RateLimiter(max_calls=20, period=3600)  # 20/hour

    async def search(
        self,
        query: str,
        max_results: int = 5,
        focus: str = "web"  # "web" | "academic" | "writing"
    ) -> dict:
        """Execute AI-powered search with citations."""
        if not self.api_key:
            return {"status": "skipped", "reason": "no_api_key"}

        # Rate limit check
        if not self.rate_limit.allow():
            return {"status": "rate_limited"}

        try:
            async with httpx.AsyncClient() as client:
                response = await client.post(
                    f"{self.base_url}/chat/completions",
                    headers={"Authorization": f"Bearer {self.api_key}"},
                    json={
                        "model": "llama-3.1-sonar-small-128k-online",
                        "messages": [
                            {
                                "role": "system",
                                "content": "Be precise and concise. Provide sources."
                            },
                            {
                                "role": "user",
                                "content": query
                            }
                        ],
                        "max_tokens": 1000,
                        "temperature": 0.2,
                        "top_p": 0.9,
                        "search_domain_filter": ["perplexity.ai"] if focus == "web" else [],
                        "return_images": False,
                        "return_related_questions": False
                    },
                    timeout=30.0
                )
                response.raise_for_status()
                data = response.json()

                return {
                    "status": "success",
                    "answer": data["choices"][0]["message"]["content"],
                    "citations": data.get("citations", []),
                    "usage": data.get("usage", {})
                }

        except Exception as e:
            return {"status": "error", "error": str(e)}


@lru_cache(maxsize=1)
def get_perplexity_client() -> PerplexityClient:
    """Get singleton Perplexity client."""
    return PerplexityClient()
```

### Step 3: Create Research Tools with Fallback Cascade

```python
# src/graph/research_tools.py

from src.integrations.perplexity_client import get_perplexity_client
from src.graph.query_interface import execute_natural_language_query

async def research_vendor(
    vendor_name: str,
    query: str,
    use_firecrawl: bool = True,
    use_perplexity: bool = True
) -> dict:
    """Research vendor with fallback cascade: Firecrawl → Perplexity → Local."""

    # Try Firecrawl first (structured web scraping)
    if use_firecrawl:
        try:
            from mcp_use import MCPClient

            client = MCPClient.from_config(".claude/mcp.json")
            await client.create_all_sessions()
            firecrawl = client.get_session("firecrawl")

            result = await firecrawl.call_tool(
                name="crawl",
                arguments={
                    "url": f"https://{vendor_name}.com",
                    "query": query,
                    "max_depth": 2
                }
            )

            if not result.isError:
                return {
                    "source": "firecrawl",
                    "status": "success",
                    "content": result.content,
                    "fallback_used": False
                }

        except Exception as e:
            logger.warning(f"Firecrawl failed: {e}, falling back to Perplexity")

    # Fallback to Perplexity (AI-powered search)
    if use_perplexity:
        try:
            client = get_perplexity_client()
            result = await client.search(
                query=f"{vendor_name}: {query}",
                focus="web"
            )

            if result["status"] == "success":
                return {
                    "source": "perplexity",
                    "status": "success",
                    "content": result["answer"],
                    "citations": result.get("citations", []),
                    "fallback_used": True
                }

        except Exception as e:
            logger.warning(f"Perplexity failed: {e}, falling back to local")

    # Final fallback: Local graph search
    try:
        result = await execute_natural_language_query(
            f"find decisions or conventions about {vendor_name}"
        )

        return {
            "source": "local_graph",
            "status": "success",
            "content": result,
            "fallback_used": True
        }

    except Exception as e:
        return {
            "source": "none",
            "status": "all_failed",
            "error": str(e)
        }


async def search_documentation(
    topic: str,
    library: str | None = None,
    year: int = 2026
) -> dict:
    """Search for technical documentation with AI assistance."""
    client = get_perplexity_client()

    query = f"{library} {topic} documentation {year}" if library else f"{topic} documentation {year}"

    result = await client.search(query=query, focus="web")

    if result["status"] == "success":
        return {
            "status": "success",
            "answer": result["answer"],
            "citations": result["citations"],
            "query": query
        }

    # Fallback to local graph
    local_result = await execute_natural_language_query(
        f"find documentation about {topic}"
    )

    return {
        "status": "fallback",
        "content": local_result,
        "query": query
    }
```

## Rate Limiting & Cost Control

**Firecrawl**:
- Free tier: 500 credits/month
- Rate limit: 10 calls/hour (configurable in `RateLimiter`)
- Cost: ~$0.001/page crawled

**Perplexity**:
- Starter tier: $20/month for 5000 queries
- Rate limit: 20 calls/hour
- Cost: ~$0.004/query

**Safety measures**:
- LRU cache on queries (1-hour TTL)
- Rate limiters prevent overuse
- Fail-open pattern (agent continues without research)

---

## Edge Cases & Validation

1. **No API keys**: Skip external tools (use local graph only)
2. **Rate limit hit**: Return cached results or empty (not error)
3. **Timeout**: Individual tool timeout 30s; entire cascade timeout 60s
4. **Invalid vendor name**: Try anyway (Perplexity may still find info)
5. **Empty results**: Valid case; return `{"status": "no_results_found"}`
