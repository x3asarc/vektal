```python
@dataclass
class ConsistencyReport:
    timestamp: datetime
    files_in_filesystem: int
    files_in_graph: int
    missing_from_graph: List[str]  # Files on disk but not in graph
    stale_in_graph: List[str]      # Files in graph but not on disk
    hash_mismatches: List[str]     # Files changed since last sync
    graph_last_sync: Optional[datetime]
    is_consistent: bool
    duration_seconds: float
```

**Implementation:**

1. **Scan filesystem:**
   ```python
   files_on_disk = set(scan_directory(".", include_dirs=["src", "tests", ".planning", "docs"]))
   ```

2. **Query graph:**
   ```cypher
   MATCH (f:File) RETURN f.path, f.content_hash, f.last_modified
   ```

3. **Compare:**
   - Files on disk not in graph -> missing_from_graph
   - Files in graph not on disk -> stale_in_graph
   - Files where content_hash differs -> hash_mismatches

**repair_divergence(report: ConsistencyReport, dry_run: bool = True) -> RepairResult**

```python
@dataclass
class RepairResult:
    files_added: int
    files_removed: int
    files_updated: int
    errors: List[str]
    dry_run: bool
```

**Repair actions:**

1. **Missing from graph -> Add:**
   ```python
   for file_path in report.missing_from_graph:
       entity = parse_and_create_entity(file_path)
       if not dry_run:
           graph.create_node(entity)
   ```

2. **Stale in graph -> Remove or mark:**
   ```python
   for file_path in report.stale_in_graph:
       if not dry_run:
           # Soft delete: mark as removed, keep for history
           graph.update_node(file_path, {"status": "removed", "removed_at": now})
   ```

3. **Hash mismatch -> Re-scan and update:**
   ```python
   for file_path in report.hash_mismatches:
       entity = parse_and_create_entity(file_path)
       if not dry_run:
           graph.update_node(file_path, entity)
   ```
