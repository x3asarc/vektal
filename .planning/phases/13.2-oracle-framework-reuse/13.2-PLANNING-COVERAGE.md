## Key Patterns Established

1. **Fail-open behavior:** All graph queries return safe defaults on unavailability
2. **Fire-and-forget emission:** All episode emissions use Celery .delay() with try/except wrapper
3. **Idempotent ingestion:** Episode IDs generated from content hash for deduplication
4. **Timeout-bounded queries:** 2s default timeout prevents request-path latency regression
5. **Contract-driven testing:** 49 tests verify behavior without live Neo4j dependency
