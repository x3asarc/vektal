## Decisions Made

1. **Neo4j 5.26 for temporal knowledge storage** - Provides native bitemporal semantics, contradiction handling, and entity deduplication vs custom implementation
2. **Graphiti-core integration** - Unified episode ingestion API faster than building custom pipeline
3. **GRAPH_ORACLE_ENABLED defaults to false** - Safe rollout pattern, prevents breaking existing flows if graph unavailable
4. **2-second timeout for all graph queries** - Prevents request-path latency regression, fail-open with fallback values
5. **Exact dependency pins** - graphiti-core==0.26.0, neo4j==5.26.0 (no >= or ~=) per project governance
