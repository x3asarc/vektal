1. Create src/jobs/graphiti_ingestor.py:

   a) Class GraphitiIngestor:
      - __init__(self, client: Optional[Graphiti] = None)
      - Accepts optional client for testing, gets from singleton if None

   b) Method ingest_episode(self, episode: dict) -> bool:
      - Validate episode has required fields (episode_type, store_id, created_at)
      - Generate episode_id for idempotency
      - Check if episode_id already ingested (simple in-memory LRU cache with 10k entries)
      - If already ingested, log debug and return True
      - Call graphiti client to add episode (async wrapper)
      - Return True on success, False on failure

   c) Method ingest_batch(self, episodes: List[dict]) -> dict:
      - Process multiple episodes
      - Return {"success": int, "failed": int, "skipped": int}

   d) Async bridge helper:
      - _run_async(self, coro) -> Any
      - Uses asyncio event loop compatible with Celery worker
      - Does NOT use asyncio.run() to avoid nested loop issues
      - Uses loop.run_until_complete() with get_event_loop_policy().get_event_loop()

   e) Timeout protection:
      - All graph writes timeout at 5 seconds
      - Timeout returns False, does not raise

   f) Logging:
      - Debug: episode ingested successfully
      - Warning: episode ingestion failed (with reason)
      - Info: batch summary

No exceptions should propagate to caller. All errors are logged and return False.
