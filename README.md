## Quick Start

```bash
pip install -r requirements.txt
cp .env.example .env
# Edit .env with your credentials
cd src && python app.py
```

## Project Structure

- `src/` - Main applications and core modules
- `tests/` - Test suite
- `config/` - Configuration files
- `data/` - Input data and resources
- `results/` - Processing results and outputs
- `docs/` - Full documentation (see [docs/INDEX.md](docs/INDEX.md))
- `archive/` - Archived code and historical artifacts

For detailed directory information, see [docs/DIRECTORY_STRUCTURE.md](docs/DIRECTORY_STRUCTURE.md).
