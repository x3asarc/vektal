# Reports Override Rules

1. Each task folder under `reports/<phase>/<task>/` must contain exactly:
   - `self-check.md`
   - `review.md`
   - `structure-audit.md`
   - `integrity-audit.md`
2. Required fields cannot be empty; use explicit `N/A` when non-applicable.
3. `review.md` must include blind-review ordering fields and final gate decision.
4. Do not delete historical reports; archive by phase if cleanup is needed.
